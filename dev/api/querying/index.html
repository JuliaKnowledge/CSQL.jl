CSQL.causal_paths — Function
causal_paths(csql; depth=2, min_score=0.0, limit=20)

Find multi-hop causal chains of length depth via self-join. For depth=2, returns paths A→B→C; for depth=3, A→B→C→D; and so on for any positive integer depth ≥ 2.

Result columns use letter aliases for nodes (a, b, c, ...) and numbered aliases for relations (r1, r2, ...), plus a path_score.

Results are ordered by path_score (sum of edge scores along the path) and filtered so that every edge meets min_score.
